# Scripts

Utility Python scripts for vault and file operations.

## Files

- `extract_hindi_pdf.py` — OCR extracts Hindi text from any PDF using Tesseract (file picker UI)
- `jpeg_to_cdi.py` — Converts a JPEG image into a raw ISO 9660 disc image (.cdr)
